
Synopsis

Tracker is a web app designed to facilitate communication and help teachers monitor student task completion. Teachers can track multiple student’s progress at a glance. It is a to do list with other party monitoring; powered by node.js, express, angular.js, and a SQL database.
 
![image](https://cloud.githubusercontent.com/assets/21045009/20155434/1a4bba7c-a689-11e6-9052-e2158d6a201c.png)
 
Motivation

This project was inspired by my own experience in a classroom trying to monitor and assist many students at once.
Key Features

•	Passport secure Login for teachers.

•	Ability to create, edit and delete: students, groups, and goals.

•	Automatic information update for student monitoring.

•	Student login and progress reporting page including option to request help on a task.

Installation

Be sure to install node dependencies to run this project.
